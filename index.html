AdTech Component Library

A Vue component library for Sky AdTech front-end applications

So why do I need this?

This library is intended to be a single place to get the baseline components needed to build a front-end web application.

The aim is for it be:

  • Sky AdTech approved styles
  • Contain all the Sky assets you need to get started
  • Unit tested with 100% coverage
  • Follow modern web development standards
  • AAA accessible

Primarily, this will be a safe starting point for any new or existing AdTech applications looking to have a modern, responsive UI.

How's it made?

Built with:

VueJs -Modern front-end UI framework

TypeScript- JavaScript superset

SCSS-CSS pre-compiler

Vite-Next generation, super fast build tool

Tested with:

Vitest-Testing framework for Vite

EsLint-Extensible JS code linting
